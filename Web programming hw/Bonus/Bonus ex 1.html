HTML & CSS Quick-Reference Guide

A handy reference for common HTML tags and CSS properties.

HTML Tags

  • <h1> - <h6>: Used for headers. The number indicates the level of importance.
  • <p>: Defines a paragraph of text.
  • <a>: Creates a hyperlink to other web pages, files, locations within the same page, or email addresses.
  • <img>: Embeds an image in the document.
  • <ul> and <ol>: Used for unordered and ordered lists, respectively.
  • <li>: Represents a list item in ordered and unordered lists.
  • <div>: Defines a division or a section in an HTML document.
  • <span>: Used for styling inline elements with CSS or grouping inline elements.
  • <input>: Creates an interactive control for entering data.
  • <button>: Represents a clickable button.

CSS Properties

  • color: Sets the color of text.
  • font-family: Specifies the font family for text.
  • background-color: Sets the background color of an element.
  • padding: Sets the padding (space) inside an element.
  • margin: Sets the margin (space) outside an element.
  • border: Sets the border around an element.
  • text-align: Specifies the horizontal alignment of text.
  • display: Specifies the type of box used for an element.
